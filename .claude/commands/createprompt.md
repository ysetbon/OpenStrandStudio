# Create Prompt

Help me create a well-structured prompt for: $ARGUMENTS

## Instructions

Analyze the request and create a comprehensive prompt that includes:

1. **Clear Objective** - Define what the prompt should accomplish
2. **Context** - Provide relevant background information
3. **Input/Output Format** - Specify expected inputs and desired output format
4. **Constraints** - List any limitations or requirements
5. **Examples** - Include example inputs/outputs if helpful

## Output Format

Provide the final prompt in a code block that can be easily copied and used.

If no arguments are provided, ask the user what kind of prompt they want to create.

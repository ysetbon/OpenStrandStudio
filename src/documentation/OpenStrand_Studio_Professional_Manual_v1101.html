    OpenStrand Studio is a specialized drawing application designed for creating rope patterns, knots, braids, and strand-based artwork. Think of it as a digital tool for drawing anything that involves interwoven or connected strands - like nautical knots, macramÃ© patterns, Celtic knots, or textile designs.

    Getting Started

    First Launch

    When you open OpenStrand Studio, you'll see:
    • A toolbar at the top with various buttons
    • A large drawing area (canvas) in the center
    • A panel on the right side for managing layers
    • Everything starts empty, ready for your first creation

    Your First Stroke

  • Click the "Attach" button (first button on the left)
  • Click and drag on the canvas to draw your first strand
  • You'll see a curved line appear - this is your first strand!

    The Drawing Canvas

    The canvas is your main workspace where you create your strand artwork.

    Grid System

    • When Grid is enabled, you'll see dots or lines
    • Strands will "snap" to these grid points for precise alignment
    • Toggle on/off with the Grid button

    Navigation

    • Zoom: Use mouse wheel to zoom in/out
    • Pan: Middle-click and drag to move around large projects
    • The canvas automatically centers your work

    Visual Feedback

    • Selected strands appear highlighted
    • Control points (small circles) show when you can adjust curves
    • Hover effects help you see what's clickable

    Layer Panel

    The right panel manages your layers (individual strands).

    Layer Buttons

    • Each strand gets a numbered button (1, 2, 3, ...)
    • Click a number to select that strand
    • Selected strands appear highlighted on the canvas

    Layer Order

    • Higher numbers appear "on top" of lower numbers
    • Drag layer buttons up/down to reorder
    • This controls which strands appear in front

    Groups

    • Combine multiple layers into groups
    • Move and organize groups together
    • Helpful for complex patterns with many parts

    Undo/Redo

    • â†¶ Undo: Go back to previous step
    • â†· Redo: Go forward after undoing
    • Every action is saved, so you can experiment freely

    Basic Workflow

  • Start with the base
    • - Click Attach mode - Draw the main loop or base strand - This becomes Layer 1

    • - Still in Attach mode - Draw additional strands that connect or cross - Each becomes a new layer (2, 3, 4...)

    • - Click Mask mode - Select strands that should go "under" others - This creates realistic weaving effects

  • Fine-tune
    • - Use Rotate or Angle/Length modes - Perfect the angles and positions - Toggle Grid for precise alignment

  • Save your work
    • - Click Save to keep your project - Click Image to export as a picture

    • Tutorial 1: Creating Your First Simple Knot

    What we're making: A basic overhand knot (the simplest knot)

    Step 1: Draw the Base Loop

  • Click Attach mode (first button)
  • Start near the left side of the canvas
  • Click and drag to create a curved line that loops around
  • End point should cross over the starting area
  • Release the mouse - you now have Layer 1

    • Result: You should see a curved line with a loop

    Step 2: Adjust the Shape

  • Click Move mode
  • Click anywhere on your strand to select it
  • You'll see small circles (control points) appear
  • Drag these circles to adjust the curve:
    • - Pull them away from the strand for sharper curves - Move them closer for gentler curves
  • Drag the strand itself to reposition it

    • Tip: Don't worry about perfection - you can always adjust later

    Step 3: Create the Knot Effect

  • Click Mask mode
  • Click on your strand where it crosses over itself
  • This creates the "over/under" effect that makes it look like a real knot

    • Step 4: Final Adjustments

  • Click Move mode again
  • Fine-tune the position and curves
  • Try Rotate mode to adjust the angle
  • Toggle Grid to help with alignment

    • Step 5: Save Your Work

  • Click Save
  • Give it a name like "My First Knot"
  • Click OK

    • Congratulations! You've created your first knot pattern.

    Tutorial 2: Making a Three-Strand Braid

    What we're making: A classic braid pattern with three parallel strands

    Step 1: Create the First Strand

  • Click Attach mode
  • Draw a gentle curve from top-left to bottom-right
  • Keep it fairly straight - this is the left strand of your braid

    • Step 2: Add the Middle Strand

  • Still in Attach mode
  • Draw another strand parallel to the first, but in the center
  • This creates Layer 2

    • Step 3: Add the Right Strand

  • Draw the third strand on the right side
  • Keep all three roughly parallel
  • Now you have Layers 1, 2, and 3

    • Step 4: Create the Braid Crossings

  • Click Attach mode again
  • Draw short connecting strands that cross between the main strands:
    • - From left strand to middle (going over) - From right strand to middle (going over) - From middle to left (going under) - From middle to right (going under)

    Step 5: Use Masks for Realistic Effect

  • Click Mask mode
  • Click on strands that should go "under" others
  • This creates the woven appearance of a real braid

    • Step 6: Clean Up with Move Mode

  • Click Move mode
  • Adjust positions so crossings look natural
  • Use control points to smooth curves
  • Make the pattern consistent along the length

    • Result: You now have a realistic-looking three-strand braid!

    Tutorial 3: Creating Over/Under Effects with Masks

    What we're making: Understanding how to make strands appear to weave over and under each other

    Understanding Masks

    • Masks make parts of strands "invisible"
    • This creates the illusion that other strands pass in front
    • Essential for realistic rope and knot effects

    Step 1: Create Two Crossing Strands

  • Attach mode: Draw one horizontal strand
  • Attach mode: Draw one vertical strand crossing the first
  • You now have an "X" shape with two layers

    • Step 2: Basic Masking

  • Click Mask mode
  • Click on the vertical strand (Layer 2)
  • Notice how part of it disappears where it crosses the horizontal strand
  • This makes the horizontal strand appear to go "in front"

    • Step 3: Switch the Effect

  • Still in Mask mode
  • Click on the horizontal strand (Layer 1) instead
  • Now the vertical strand appears to go in front
  • Click the same strand again to remove the mask

    • Step 4: Practice with Multiple Strands

  • Add a third strand diagonally across both
  • Use Mask mode to create a pattern:
    • - Strand 1 over Strand 2 - Strand 2 over Strand 3 - Strand 3 over Strand 1
  • This creates a realistic woven effect

    • Step 5: Fine-tuning Masks

  • Use Move mode to adjust positions
  • The mask effects stay with the strands as you move them
  • Use Mask mode again to add or remove effects

    • Key Point: Masking is what makes your drawings look like real rope work instead of just lines on paper.

    • Step 5: Create a Rainbow Braid

  • Create 5 parallel strands

    Tutorial 5: Using the Grid for Precise Alignment

    What we're making: A perfectly symmetrical knot pattern using the grid system

    Step 1: Enable the Grid

  • Click the Grid button in the toolbar
  • You'll see dots or lines appear on the canvas
  • These help you align strands precisely

    • Step 2: Understanding Grid Snapping

  • Attach mode: Try drawing a strand
  • Notice how it "snaps" to grid points
  • This helps create neat, professional-looking patterns
  • The snapping works for start points, end points, and curves

    • Step 3: Create a Symmetric Pattern

  • Draw the first strand from one grid point to another
  • Count the grid squares (e.g., 6 squares long)
  • Draw a second strand the exact same length
  • Use the grid to ensure perfect symmetry

    • Step 4: Grid Settings

  • Settings â†’ Drawing Settings â†’ Grid
  • Adjust grid size:
    • - Smaller = more precision - Larger = easier to see
  • Toggle "Snap to Grid" on/off as needed

    • Step 5: When to Use the Grid

    Good for:
    • Geometric patterns
    • Repeated elements
    • Professional diagrams
    • Teaching materials

    Not needed for:

    • Organic, natural-looking ropes
    • Artistic, flowing designs
    • Quick sketches

    Tutorial 6: Managing Multiple Layers

    What we're making: A complex knot with proper layer organization

    Understanding Layer Numbers

    • Each strand gets a number (1, 2, 3...)
    • Higher numbers appear "on top"
    • Layer panel shows all your strands

    Step 1: Create Multiple Strands

  • Create 6-8 strands in a complex pattern
  • Notice how they're numbered in the layer panel
  • Each number is a clickable button

    • Step 2: Selecting Layers

  • Click number 3 in the layer panel
  • Layer 3 becomes highlighted on the canvas
  • This is much easier than clicking on the canvas for complex patterns

    • Step 3: Reordering Layers

  • In the layer panel, drag a layer button up or down
  • Watch how the strand moves forward or backward
  • Use this to control which strands appear in front

    • Step 4: Hiding Layers Temporarily

  • Look for visibility controls in the layer panel
  • Hide some layers while working on others
  • This reduces visual clutter in complex projects

    • Step 5: Layer Organization Strategy

    For complex projects:
  • Bottom layers: Background elements, base structure
  • Middle layers: Main pattern elements
  • Top layers: Details, highlights, finishing touches

    • Naming convention (if available):

    • Layer 1: "Base Loop"
    • Layer 2: "Cross Over"
    • Layer 3: "Detail Strand"

    Tutorial 7: Creating Groups and Organization

    What we're making: A complex pattern organized into manageable groups

    Step 1: Create a Multi-Part Pattern

  • Create one section with 3-4 strands (e.g., a knot)
  • Create another section with 3-4 strands (e.g., a braid)

  • Select multiple layers that belong together:

  • Create multiple groups for different parts
  • Work on one group at a time
  • Hide other groups while focusing
  • This keeps large projects manageable

    Advanced Knot Construction Techniques

    The Professional Approach to Knot Drawing

    Phase 1: Foundation Structure

  • Identify the core loop: Every knot has a basic underlying structure
  • Draw the primary path: Create the main rope path without worrying about crossings
  • Mark crossing points: Note where the rope crosses over itself
  • Plan the layer hierarchy: Which sections go over/under others

    • Phase 2: Systematic Construction

  • Build from bottom up: Start with strands that go "under" everything else
  • Add middle layers: Strands that have some over/under relationships
  • Finish with top layers: Strands that go "over" most other elements
  • Refine systematically: Work through each layer, perfecting curves and positions

    • Advanced Masking Strategies

    Complex Over/Under Patterns:

  • Map the crossings: Draw a simple diagram showing every crossing point
  • Number the crossings: 1, 2, 3... to track which you've completed
  • Apply masks systematically: Work through your numbered list
  • Test the logic: Follow the rope path to ensure it's physically possible

    • Multiple Rope Knots:

  • Color-code during construction: Use temporary colors to track different ropes
  • Build one rope at a time: Complete all layers for rope 1, then rope 2, etc.
  • Coordinate the interactions: Plan how different ropes cross each other
  • Unify colors at the end: Change all ropes to final colors when structure is complete

    • Creating Publication-Quality Diagrams

    Technical Illustration Standards

    Consistency Requirements:

    • Uniform strand width: All ropes of same type must be identical thickness
    • Consistent color coding: Same colors mean same things throughout
    • Standardized spacing: Equal gaps between parallel strands
    • Aligned elements: Use grid religiously for professional appearance

    Visual Hierarchy:

  • Primary elements: Main rope structure - highest contrast
  • Secondary elements: Labels, arrows - medium contrast
  • Background elements: Grid, reference lines - low contrast
  • Annotations: Clear, readable, consistently positioned

    • Creating Step-by-Step Instruction Diagrams

    Multi-Stage Workflow:

  • Stage 1 - Setup: Show initial rope positioning
  • Stage 2 - First moves: Highlight what changes from Stage 1
  • Stage 3 - Progression: Continue showing changes
  • Stage 4 - Final result: Complete knot with all details

    • Highlighting Changes Between Steps:

  • Use color coding: Bright colors for "new" elements each step
  • Add directional arrows: Show movement and direction clearly
  • Fade previous steps: Gray out unchanged elements to focus on new parts
  • Number the sequences: Clear numbering for step-by-step progression

    • Export Strategies for Different Uses

    Output Optimization by Use Case

    Web Display (Screen):

    • Resolution: 72-96 DPI sufficient
    • Format: PNG with transparency
    • Size: Optimize for common screen sizes
    • Colors: RGB color space

    Print Publication:

    • Resolution: 300 DPI minimum, 600 DPI for line art
    • Format: PNG or PDF
    • Size: Exact dimensions in inches/cm
    • Colors: CMYK color space for professional printing

    Educational Posters:

    • Resolution: 150-300 DPI depending on viewing distance
    • Format: PDF for scalability
    • Size: Large format (11x17", 18x24", etc.)
    • Colors: High contrast for classroom visibility

    Technical Documentation:

    • Resolution: 300 DPI for crisp lines
    • Format: Vector PDF when possible
    • Size: Standard document sizes (A4, Letter)
    • Colors: Black and white or limited color palette

    Troubleshooting Encyclopedia

    Startup and Installation Issues

    Application Won't Start

    Windows specific:
    • Error: "DLL load failed while importing QtWidgets"
    - Cause: PyQt5 DLL conflicts with other installed software - Solution: Use the build_with_venv.bat script for clean environment - Prevention: Install in virtual environment

    • Error: "Python not found"
    - Cause: Python not in system PATH - Solution: Reinstall Python with "Add to PATH" option checked - Alternative: Use full path to python.exe

    Drawing and Interface Issues

    Drawing Problems

    Strands don't appear when drawn:
    • Check: Are you in Attach mode?
    • Check: Is canvas zoomed out too far to see details?
    • Solution: Switch to Attach mode, zoom in to see drawn strands

    Can't select strands:

    • Check: Are you in Move or Select mode?
    • Check: Are layers hidden?
    • Solution: Use layer panel numbers for precise selection
    • Alternative: Switch to Select mode for easier clicking

    File Operation Issues

    Saving Problems

    Can't save files:
    • Check: Write permissions to target folder
    • Check: Available disk space
    • Solution: Save to Desktop or Documents folder
    • Alternative: Run as administrator (Windows)

    Files corrupted after save:

    • Cause: Disk space ran out during save
    • Cause: System crash during save operation
    • Prevention: Check disk space before saving
    • Solution: Use multiple backup saves

    Loading Problems

    Files won't open:
    • Check: File extension (.json for projects)
    • Check: File not corrupted (try backup copy)
    • Solution: Try loading a sample file first
    • Workaround: Start new project, copy elements manually

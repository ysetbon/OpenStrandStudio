Layer Context Menu — Arrow Section (v1.109)

Interactive mock of what shipped on branch worktree-setting-dialog-upgrade-109: themed dropdowns with proper triangle arrows, 50% larger checkboxes, and the new Arrow Sizes → Adjust dropdown exposing all six settings-dialog arrow numbers with the segmented [ − | value | + ] steppers. Everything is clickable — try the dropdowns, toggles and steppers (press-and-hold repeats).
In the app: right-click a layer button → enable Show Full Arrow → the arrow section appears with this exact structure. Stepping a value in Arrow Sizes redraws the canvas live, syncs the Settings dialog spin boxes, and saves to the settings file when the popup closes.
